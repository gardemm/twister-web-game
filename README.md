### What is this repository for? ###

Make **twister** web game for fun

* 1.0.0 (beta)

### How do I get set up? ###

* Dependencies
    - `node >=0.12.0`  | `gulp ^3.9.0` 

* **How to run tests**
    - gulp serve:test

* **Deployment instructions**
    - use `coffescript` to generate js
    - gulp serve (automaticaly generate *.js from *.coffee)
    - gulp serve:dist
